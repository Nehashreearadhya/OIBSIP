## 📂 Project Tasks

### Task 1: Iris Flower Classification
**Description:** Developed a Machine Learning model to classify Iris flowers into three species (*Setosa, Versicolor, and Virginica*) based on sepal and petal measurements.

**Key Features:**
* **EDA:** Performed Exploratory Data Analysis using Plotly for interactive visualizations.
* **Model:** Implemented the **K-Nearest Neighbors (KNN)** algorithm.
* **Accuracy:** Achieved high precision in classifying species.

**Tech Stack:**
* Python
* Scikit-learn
* Pandas & NumPy
* Plotly / Seaborn
